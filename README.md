## About

This is the issues and releases repository for [Loophole](https://getloophole.in), the AI IDE that thinks while you code.

Loophole is:

- An AI-powered code editor built for developers who want deep, context-aware intelligence — not just autocomplete
- Designed to support multiple AI models so you can use the right one for the task at hand
- Available on Windows, macOS, and Linux

## Installation

You can [download Loophole](https://github.com/loophole-ai/loophole/releases) and read the [docs](https://github.com/loophole-ai/loophole/wiki) for platform-specific instructions.

## Changelog and Releases

We publish releases on the [Releases page](https://github.com/loophole-ai/loophole/releases). Each release includes platform-specific installers for Windows, macOS, and Linux.

## Issues, Bugs, and Feature Requests

Please [search through existing issues](https://github.com/loophole-ai/loophole/issues?q=is%3Aissue+is%3Aopen+sort%3Areactions-%2B1-desc) before filing a new one — your bug or request may already be tracked.

If you can't find it, [open a new issue](https://github.com/loophole-ai/loophole/issues/new/choose) using one of the provided templates. This helps us track and prioritize effectively.
